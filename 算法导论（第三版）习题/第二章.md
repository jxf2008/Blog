## 2-1-2
C++实现
```c++
const int Len = 6;
int A[Len] = {1,8,5,3,9,3};


for(int i = 1 ; i < Len ; ++i){
    for(int j = i-1 ; j >= 0 ; --j){
	    if(A[j+1]  > A[j]){
		    int tmp = A[j+1];
		    A[j+1] = A[j];
		    A[j] = tmp;
	    }else
		    break;
	}
}
```

1. 将A[n-1]和B[n-1]和C[n]相加。令k=n
2. 判断C[k]是否大于1，如果大于1，将C[k-1]和(C[k]-2)相加，并将C[k]置为C[k]-2
3. --k,然后重复步骤2，直到C[k-1]小于或等于1，或者k=0
4. --n,重复步骤1-3,直到n=0

伪代码实现
```python
for n = A.size() - 1 to 0
    C[n+1] = A[n] + B[n] + A[n+1]
    k = n+1
    while k>=0 && C[k]>1  #步骤3的判断
        C[k-1] = C[k-1] + (C[k]-2) 
        C[k] = C[k] - 2
        --k
```

| 最小时间 | 最大时间 |
| :----: | :----: |
| T1 * n | T1 * n |
| T2 * Σ（1 to n) | T2 * Σ（1 to n) |

1. T1 * n + (T2+T3+T4) * (n^2-1) / 2
